# Railway Concession System - Frontend

A modern React-based frontend for the Railway Concession Management System.

## Features

- **Student Portal**: Apply for concessions, track status, manage profile
- **Staff Portal**: Manage applications, students, generate reports
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Real-time Updates**: Live application status tracking
- **Secure Authentication**: Role-based access control

## Tech Stack

- **Frontend**: React 18 + Vite
- **Routing**: React Router v6
- **Styling**: Tailwind CSS
- **HTTP Client**: Axios
- **State Management**: React Context + Custom Hooks

## Prerequisites

- Node.js 16+ 
- npm or yarn
- Backend API server running on port 8080
